# Week 2：一阶 / 二阶系统 / 状态空间

**目标**：从时域和频域两个角度理解经典动态系统，能用 Transfer Fcn 搭一阶 / 二阶模型并解释响应特征，建立状态空间概念。

## 模型清单

| 文件 | 内容 | 关键概念 |
|------|------|---------|
| `first_order_comparison.slx` | 一阶系统不同时间常数 T 的阶跃响应 | 时间常数 T，63.2% 稳态 |
| `second_order_comparison.slx` | 二阶系统不同阻尼比 ζ 的响应 | 欠 / 临界 / 过阻尼 |
| `natural_frequency_comparison.slx` | 不同固有频率 ωn 的响应对比 | 固有频率 ωn |
| `mass_spring_damper.slx` | 质量-弹簧-阻尼物理模型 | 集总参数建模 |

## 知识点

- **一阶系统**：G(s) = 1 / (T·s + 1)
  - T 决定响应快慢；t = T 时达到 63.2% 稳态值（1 − e⁻¹）
- **二阶系统**：G(s) = ωn² / (s² + 2ζωn·s + ωn²)
  - ζ < 1 欠阻尼（振荡衰减）；ζ = 1 临界阻尼；ζ > 1 过阻尼（无超调）
  - ωn 决定固有频率，ζ 决定振荡衰减速度
- **状态空间**（理论）：ẋ = A·x + B·u，y = C·x + D·u
  - 多输入多输出的统一描述形式，是后续现代控制 / MPC 的基础
